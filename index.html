Multi-Engine Fusion Architecture

RuiQi WAF leverages a sophisticated architecture that combines high-performance proxy capabilities with deep security inspection:

HAProxy Frontend Layer

High-performance HTTP/HTTPS proxy with TLS 1.3 support, basic filtering, and intelligent traffic distribution.

Coraza WAF Core Engine

Next-generation open-source WAF engine compatible with ModSecurity, supporting OWASP CRS and custom rules.

SPOE Communication

Efficient communication protocol between HAProxy and Coraza, optimizing detection performance.
